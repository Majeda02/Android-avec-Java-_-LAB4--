## "# Android-avec-Java-_-LAB4--" 

### LAB 4 – Convertisseur Température & Distance (Fragments + Onglets)

#### Objectif général
Réaliser une application Android composée de deux onglets :
* un pour convertir les températures (°C ↔ °F)
* un autre pour convertir les distances (Km ↔ Miles)
  
L’application doit  intercepter la touche Retour pour demander confirmation avant de fermer.
